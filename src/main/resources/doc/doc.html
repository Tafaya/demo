Demo project endpoint documentation


GET /doc

Prints out this documentation


GET /getUnique

Collect unique file names with extension ext on the given path

  Parameters
  String path - the root path of the search
  String ext - files with this extension will be considered during the search

GET /history

Returns the list of previous searches


POST /gen

Generates a sample folder structure and files with the given extension ext and files not with this extension for testing purposes.
Subsequent calls are not recreating the folder structure. The output is the file structure 

  Parameters
  String ext - extension for the test files
